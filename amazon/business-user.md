## Giai Đoạn 6: Best Practices

### Effective Prompting

```
Good prompts:
✓ "Summarize the Q4 financial report, focusing on revenue growth"
✓ "Find our data retention policy for European customers"
✓ "Compare project A and B timelines"

Avoid:
✗ "Tell me everything about sales"
✗ "What should I do?"
✗ Single word queries
```

### Security Awareness

- [ ] Understand data access controls
- [ ] Don't share sensitive info inappropriately
- [ ] Verify answers for accuracy
- [ ] Report incorrect responses
