## Giai Đoạn 1: Gemini Fundamentals

### Free Tools

| Tool | Features | Access |
|------|----------|--------|
| Gemini (gemini.google.com) | Chat, generate | Free |
| AI Studio | Prompt testing | Free |
| NotebookLM | Document analysis | Free |

### Getting Started

- [ ] Tạo Google account
- [ ] Truy cập gemini.google.com
- [ ] Thử nghiệm các prompts cơ bản
- [ ] Hiểu multimodal (upload images)

### Basic Prompting

```
Effective prompt structure:
1. Role: "As a marketing expert..."
2. Task: "...analyze this campaign data"
3. Context: "...for a B2B SaaS company"
4. Format: "...in bullet points"
5. Constraints: "...focus on ROI metrics"
```

## Giai Đoạn 3: NotebookLM

### Getting Started

1. Go to notebooklm.google.com
2. Create new notebook
3. Upload sources (docs, PDFs, websites)
4. Start asking questions

### Key Features

| Feature | Description |
|---------|-------------|
| Source grounding | Answers from your docs |
| Citations | See where info comes from |
| Audio Overview | Generate podcast |
| FAQ generation | Auto create Q&A |
| Study Guide | Learning materials |

### Use Cases

```
Research:
- Upload research papers
- Ask synthesis questions
- Generate summaries

Business:
- Upload reports, policies
- Quick answers
- Create training materials

Learning:
- Upload textbooks
- Create study guides
- Test knowledge
```

### Audio Overview

- [ ] Upload multiple sources
- [ ] Click "Generate audio"
- [ ] Listen to AI-generated discussion
- [ ] Share with team

## Tips & Best Practices

### Prompting Tips

1. **Be specific**: "Summarize Q4 sales report highlighting declining regions"

2. **Use context**: Upload relevant files before asking

3. **Iterate**: Refine responses with follow-up

4. **Verify**: Always check AI outputs for accuracy

### Privacy Considerations

- [ ] Understand data handling
- [ ] Don't share sensitive info unnecessarily
- [ ] Use enterprise controls if available
- [ ] Review outputs before sharing
