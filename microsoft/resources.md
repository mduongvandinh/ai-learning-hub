## Chứng chỉ

### Lộ trình đề xuất

```
Cấp 1 (Nền tảng):
├── AI-900: Azure AI Fundamentals
└── MS-900: Microsoft 365 Fundamentals

Cấp 2 (Associate):
├── AI-102: Azure AI Engineer
├── DP-100: Azure Data Scientist
└── MS-721: M365 Copilot

Cấp 3 (Expert):
└── AZ-305: Azure Solutions Architect
```

### Mẹo thi

1. **AI-900**: Tập trung vào tổng quan dịch vụ, không phải triển khai
2. **AI-102**: Labs thực hành rất quan trọng

4. **MS-721**: Hiểu về quản trị Copilot
